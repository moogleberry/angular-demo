Angular Demo

This is a demonstration of key concepts in Angular JS 1.x:

  • Two-way Binding
  • Built-in UI Tools
  • Angular Directives
  • Angular Services
  • Tricky Javascript Inheritance
  • Jasmine Unit Testing

This demo is intended for audiences who have:

  • Some understanding of HTML/CSS
  • Little to no understanding of Angular
  • A desire to learn Angular

This demo uses:

  • AngularJS 1.4.7
  • Bootstrap
